# personal-website

A static personal website built with SvelteKit.

## Tech Stack

- **Framework**: SvelteKit v2 / Svelte v5
- **Styling**: Tailwind CSS v4
- **Language**: TypeScript 5.9
- **Typography**: System-local font stacks, no bundled webfonts
- **Testing**: Playwright (E2E)
- **Deployment**: GitHub Pages (adapter-static)

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (LTS recommended)
- [pnpm](https://pnpm.io/)

### Commands

```bash
pnpm install   # Install dependencies
pnpm dev       # Start dev server
pnpm build     # Production build
pnpm preview   # Preview production build
pnpm test      # Run E2E tests
pnpm lint      # Check formatting & lint
pnpm format    # Auto-format code
```
